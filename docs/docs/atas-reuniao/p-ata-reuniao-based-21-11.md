### 2. Integração do código e branches

**Discussão:** Foram esclarecidas dúvidas sobre o uso das branches (experiment, front, integração) e a necessidade de organizar melhor o fluxo de merges para evitar conflitos.

**Decisão:** Adotar o fluxo acordado de criação de branches a partir de versões estáveis, abrir PRs para revisão e só então integrar nas branches principais.
