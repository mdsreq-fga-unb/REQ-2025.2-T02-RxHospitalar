# Checklist de avaliação do USM do grupo Berkanan
Os itens que cumprem os critérios foram assinalados.

- [ ] O USM identifica claramente as personas e seus objetivos 
- [ ] A jornada do usuário está completamente mapeada, cobrindo todas as grandes etapas do processo
- [ ] As atividades principais (backbone) estão evidentes e bem organizadas
- [ ] As histórias de usuário estão descritas do ponto de vista do usuário, apresentando benefícios claros
- [ ] O USM mostra prioridades (releases, MVP, incrementos visíveis)
- [ ] As histórias e entregas estão alinhadas aos objetivos do negócio
- [ ] O USM permite identificar lacunas e dependências entre histórias
